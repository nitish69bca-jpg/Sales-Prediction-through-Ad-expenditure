# Sales Prediction Through Advertising Expenditure

## Project Overview
This project focuses on predicting product sales based on advertising expenditure across different marketing channels.  
The goal is to understand how spending on advertising influences sales and to build a machine learning model that can make accurate sales predictions.

---

##  Objectives
- Analyze the relationship between advertising spend and sales  
- Identify which advertising channels impact sales the most  
- Build a predictive model for sales forecasting  
- Evaluate the performance of the model using appropriate metrics  

---

##  Dataset Description
The dataset contains information about advertising budgets and corresponding sales figures.

### Features:
- **TV Advertising Spend**
- **Radio Advertising Spend**
- **Newspaper Advertising Spend**

### Target Variable:
- **Sales**
